        } else {
            $this->output("Failed!");

    private function read() {
        $res = "";
        if(isset($this->filename)) {

        if($this->op === "2")

基本思路就是构造op=2,file=flag.php去读文件

有两个地方需要绕过一下

第一,因为是protected属性，所以在序列化的时候会自动加送%00,但是在is_valid函数中会逐子判断ascii必须>=32或<=125

__construct方法，接收参数$file，$this->source = $file将$file的值赋给source。而$file的值直接就是inde.php，所以还是输出index.php。

__toString方法，当前对象访问str再访问source，然后返回这个值，就是把类当作字符串使用时触发

wakeup方法，使用反序列化的时候触发，而这里同时也可以触发toString方法。里面有waf，过滤一些协议，但是显而易见可以用filter

6.反序列化函数会触发wakeup方法
