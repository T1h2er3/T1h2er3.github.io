BUUCTF刷题2

[护网杯 2018]easy_tornado

访问页面，有三个链接

/flag.txt

/welcome.txt

/hints.txt

很明显，flag在fllllllllllllag,要想访问fllllllllllllag必须是filename=/fllllllllllllag+md5(cookie_secret+md5(fllllllllllllag))

这里我们只需要知道这个cookie_secret的值，就可以访问了

welcome.txt有个render，会不会和模板有关

先尝试不用filehash访问一下/fllllllllllllag

这里很明显有个模板注入

那么思路应该很清晰，就是利用模板注入，根据题目的意思应该是利用这个获取cookie_secret的值

查阅关于模板注入的知识

初步判断是flask的模板

查询关于flask模板关于cookie_secret的知识

查阅了大部分资料

发现了handler.setting里存在cookie_secret

拿到之后算出filehash的值即可

<?php
$filename='/fllllllllllllag';
$secret='f555540d-d7fb-4284-9eb8-3dd8f4f27a6d';
echo md5($secret.md5($filename));
?>

拿到flag

简单的反序列化，拿到flag的条件是username=admin&password=100

序列化的结果为

O:4:”Name”:2:{s:14:”Nameusername”;s:5:”admin”;s:14:”Namepassword”;i:100;}

要修改为

O:4:”Name”:3:{s:14:”%00Name%00username”;s:5:”admin”;s:14:”%00Name%00password”;i:100;}

将2改为3，绕过weak_up函数

加%00是因为private变量有空格复制会丢失

[极客大挑战 2019]Http

访问页面有个Secret.php

进去之后要求你必须来自https://www.Sycsecret.com

添加 Referer: https://www.Sycsecret.com之后

有要求用Syclover浏览器访问

修改 User-Agent: Syclover后

要求本地访问

添加 X-Forwarded-For:127.0.0.1

拿到flag

其实有的题可能还要求访问时间必须是2099年呀之类的

就是添加If-Unmodified-Since:2099
